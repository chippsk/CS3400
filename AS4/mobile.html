Developing Websites for Mobile Devices

Guidelines

Mobile devices are on the rise, and year after year more mobile devices connect to the internet and browse websites. It is imperative that website designers account for mobile devices. Here is some guidelines to developing for mobile devices.

Google recommends that when developing for mobile devices to use HTTP redirection with status code 301 or 302. Or to use javascript redirects.
Use video tags to have videos, mobile devices generally don't have Flash and things like it.
Limit useage of smartphone 404's. Even if the URL is a desktop only, redirect to a different page rather than kill the browsing all together.
Page speed is of the utmost importance. It is possible to create two different sites, on geared to toward mobile devices that way you can maximize your effiecieny.
