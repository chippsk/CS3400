Design Rules

Human Factors

  • Consistant Load Times
  • Mouse Speed
  • Updating User
  • Results for every action

These are human factors associated with user interface design. Things that may not make sense from a technological standpoint, but do make sense when we remember that human beings have to use our interfaces.

High Level Factors

8 Golden Rules of Interface Design by Ben Shneiderman

  • Strive for consistency
  • Enable frequent users to use shortcuts
  • Offer informative feedback
  • Design dialog to yield closure
  • Offer simple error handling
  • Permit easy reversal of actions
  • Support internal focus of control
  • Reduce short-term memory load

Ben Shneiderman wrote 8 Golden Rules for user interface design. These rules if followed generally strive toward a more complete user interface that users will like to use and return to.
