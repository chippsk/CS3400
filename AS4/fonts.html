Font Type

Choosing a font type can be an art. There is no shortage of types of fonts to use, so lets outline what to think about when choosing font types.
Establish a goal
The type of fonts a website uses will be one of the first things a user will notice when browsing to a site. Things to think about are what the site will be used for. There are fonts for everything. Sites that have extensive reading, like large articles, need to use fonts that are easy to read and very legible. However sites that are meant for entertainment, such as advertising, fonts that are more decorative can be used.
Legibility and readibility
Legible and readible fonts generally have characteristics such as, convential letterforms (no artsy stuff) and proper spacing between characters.
Limit the amount of typefaces per site. Generalize around a theme. Putting too many typefaces in a site will make it chaotic.
