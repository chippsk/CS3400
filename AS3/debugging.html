Debugging Javascript.

You should have been faced with some strange behaviour of your code somewhere in time. When it happens it is Debugging time. Most of the major browser nowadays come with enough tools to help you in this task. And there are also some specific tools for you to do it.

When the time comes you search for the developer tools in the browser you're developing to. If you need something more elaborate you might want to give a try in the Firebug tool, they even let you use breakpoints!
