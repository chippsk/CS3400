Canvas

Canvas is a HTML element that is used to draw graphics. In order to draw something in the canvas you must use javascript.

Let's see an example:

content

The code needed above was:

<script type="text/javascript"> var canvas = document.getElementById('exampleCanvas'); var ctx = canvas.getContext('2d'); function getPos(canvas, event){ var rect = canvas.getBoundingClientRect(); return { x: event.clientX - rect.left, y: event.clientY - rect.top }; }; // logic to draw. canvas.addEventListener('mousemove', function(event) { var position = getPos(canvas, event); canvas.fillStyle = '#'+(Math.random().toString(16) + '000000').slice(2, 8); ctx.fillRect(position.x, position.y, 1, 1); }, false); </<script>
