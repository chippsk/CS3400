Unit Testing in Javascript

If you need to do some unit testing to make your code more robust and trustworthy you might want to do some Unit Testing to help you in this task you can use QUnit.

This may take a while

This should go in the head to include the qunit framework <head> <meta charset="utf-8"> <title>QUnit basic example</title> <link rel="stylesheet" href="http://code.jquery.com/qunit/qunit-1.15.0.css"> </head> Then actually using it would look something like this. <script src="http://code.jquery.com/qunit/qunit-1.15.0.js"></script> <script> QUnit.test( "a basic test example", function( assert ) { var value = "hello"; assert.equal( value, "hello", "We expect value to be hello" ); }); </script>
